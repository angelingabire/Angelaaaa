DATABASE AND JAVA WITH BANKING MANAGEMENT SYSTEM

  1. DATABASE

    2. Database name used: banking_management_system.

    • Tables

    • account Table

      • account Table This table stores and manages detailed information about individual customer accounts, such as their account identification, customer information, account balance, transaction history, account status and account type


    • branch table.

      • branch Table This table stores information about the different branches of a bank . This includes data such as the branch name, branch code, address, contact information, manager details, operating hours and any other relevant information.


    • customer table

      • customer Table This table contains information about the customers of the bank. This includes data such as customer name ,address, contact information ,account details, transaction history and any other relevant information.


    • employee table

      • employee table This table records information about the bank’s employees. This includes data such as employee name, employee id, contact information, job title, department, salary, and any other relevant information related to the employees working at the bank

    • transaction table

      • transaction table This table plays a crucial role in recording all financial transactions that occur within the bank. The table stores details about all types of transactions ,such as deposits ,withdrawals ,transfers, payments, and other financial activities conducted by the bank’s customers.
